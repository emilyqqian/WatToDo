# Project Charter

## project title + date
- To do app - October 7th, 2025
## Purpose
- To design and implement a simple yet effective web-based To-Do application that allows users to manage their daily tasks from any device. The application will initially run locally using Flask and a MySQL database, with the goal of transitioning to a fully web-accessible system.
## Objectives
- Enable users to create, update, and manage tasks in a persistent database.
- 
## Scope
- Development of a local-only web-based To-Do application using Flask and MySQL. 
- Implementation of core CRUD operations (`add`, `update`, `delete`) and task retrieval functions (`next`, `today`, `tomorrow`).  
- User-specific task retrieval based on `userid`.  
- Comprehensive test plan and test cases for all functions.

## Deliverables
- MySQL Database
- Test plan + test cases
- User stories and use cases
- web page
- back end
